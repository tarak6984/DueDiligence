## üìù How to Add Screenshots

To add screenshots to this directory:

1. **Take Screenshots**
   - Use your preferred screenshot tool
   - Capture key features and workflows
   - Use consistent window size (1920x1080 recommended)

2. **Save Files**
   - Save screenshots in this directory
   - Use descriptive names matching the titles above
   - Recommended format: PNG for quality, JPG for smaller size

3. **Update Links**
   - Replace placeholder paths with actual filenames
   - Keep the markdown format: `![Alt Text](./filename.png)`

**Tools:**
- Windows: Snipping Tool, Windows + Shift + S
